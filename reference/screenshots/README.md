# لقطات المرجع والمقارنة

ضع هنا لقطات **full-page** من:

1. `https://amarabijouxx.youcan.store/` وصفحات الفئات والمنتج.
2. نفس الصفحات على `http://localhost:3000` (أو المنفذ الذي تستخدمينه).

الاسم المقترح: `reference-home.png`، `clone-home.png`، إلخ.

يمكن أخذ اللقطات من أدوات المتصفح أو من Cursor IDE browser بعد تشغيل `npm run dev`.
